# 项目返回代码含义

### Error Code
`-1` 当前邮箱账号已存在 </br>
`-2` 当前邮箱账号不存在 </br>
`-3` 密码错误 </br>
`-4` 邮箱错误

### Success Code
`1` 用户注册成功 </br>
`2` 用户登录成功 </br>
`3` 用户成功删除账号

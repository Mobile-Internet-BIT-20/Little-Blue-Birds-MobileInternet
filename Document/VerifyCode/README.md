# 项目返回代码含义

### Error Code
`-1` 当前邮箱账号已存在 </br>
`-2` 当前邮箱账号不存在 </br>
`-3` 密码错误 </br>
`-4` 邮箱错误 </br>
`-5` 当前账号不存在

### Success Code
`1` 用户注册成功 </br>
`2` 用户登录成功 </br>
`3` 用户成功删除账号 </br>
`4` 成功请求用户的所有信息 </br>
`5` 成功请求访客视角用户信息 </br>
`6` 成功请求用户所发布的帖子 </br>
`7` 用户更新基本信息完成 </br>
`8` 成功更新用户电子邮箱
